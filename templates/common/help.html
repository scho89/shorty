Domain setup guide

Use this page when registering a new short-link domain and verifying ownership before it goes live.

Step 1

Add a domain

Open Domains, enter the host you want to use, and submit it. Shorty will create a verification token for that domain.

  1. Go to the Domains page.
  2. Enter the host name you want to use, such as t.example.com.
  3. Select Register domain.
Step 2

Prove ownership

After registration, Shorty shows a DNS TXT verification value in the domain registry. Add that exact value to your DNS zone and then run verification from the app.

  1. Copy the TXT value shown in the Status column.
  2. Create a DNS TXT record for the same host.
  3. Wait for DNS propagation.
  4. Select Verify in the domain registry.
Step 3

Set the CNAME record

The short-link host must also point to the current routing target below.

Record type: CNAME
Target: {{ cname_target }}

Example: if the short domain is t.example.com, create a CNAME for t that points to {{ cname_target }}.

Checklist

Before publishing links

  • The domain is listed on the Domains page.
  • The TXT ownership record has been added.
  • The CNAME points to {{ cname_target }}.
  • The status shows Verified.
  • You can then create short links from the dashboard.
